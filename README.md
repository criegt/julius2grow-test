# julius2grow-test

Prueba esta desarrollada utilizando: 
- `.NET 5`
- `Node.js v12.19+`
- `Angular CLI v10.2+`

## Pasos para ejecutar
- Clonar repositorio
```
git clone https://github.com/criegt/julius2grow-test
cd julius2grow-test
```

- Ejecutar API de .NET
```
dotnet run -p .\src\Julius2GrowTest.WebApi\        
```

- Ejecutar cliente en Angular
```
cd .\client\julius2grow-client\  
npm i
ng serve
```
